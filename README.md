# CycleGAN: I'm Something of a Painter Myself
### Unpaired Image-to-Image Translation (Photo to Monet)

рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃБрЃџрЃўрЃљ Kaggle-рЃўрЃА рЃерЃћрЃ»рЃўрЃЉрЃарЃўрЃА рЃцрЃљрЃарЃњрЃџрЃћрЃЉрЃерЃў. рЃЏрЃўрЃќрЃљрЃюрЃўрЃљ рЃерЃћрЃўрЃЦрЃЏрЃюрЃљрЃА **Generative Adversarial Network (GAN)**, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЕрЃЋрЃћрЃБрЃџрЃћрЃЉрЃарЃўрЃЋ рЃцрЃЮрЃбрЃЮрЃћрЃЉрЃА рЃњрЃљрЃЊрЃљрЃљрЃЦрЃфрЃћрЃЋрЃА рЃЎрЃџрЃЮрЃЊ рЃЏрЃЮрЃюрЃћрЃА рЃАрЃбрЃўрЃџрЃўрЃА рЃюрЃљрЃ«рЃљрЃбрЃћрЃЉрЃљрЃЊ, **Paired Data**-рЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃњрЃљрЃарЃћрЃерЃћ.

## Methodology & Architecture

рЃљрЃЏрЃЮрЃфрЃљрЃюрЃўрЃА рЃњрЃљрЃЊрЃљрЃАрЃљрЃГрЃарЃћрЃџрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћрЃЌ **CycleGAN** рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ, рЃарЃљрЃЊрЃњрЃљрЃю рЃўрЃА рЃўрЃЊрЃћрЃљрЃџрЃБрЃарЃўрЃљ **Unpaired Image-to-Image Translation**-рЃўрЃАрЃЌрЃЋрЃўрЃА. рЃЏрЃЮрЃЊрЃћрЃџрЃў рЃАрЃгрЃљрЃЋрЃџрЃЮрЃЉрЃА рЃЮрЃарЃЏрЃљрЃњ рЃбрЃарЃљрЃюрЃАрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА ($G: A \to B$ рЃЊрЃљ $F: B \to A$) рЃЊрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА **Cycle Consistency Loss**-рЃА, рЃарЃљрЃЌрЃљ рЃерЃћрЃўрЃюрЃљрЃарЃЕрЃБрЃюрЃЮрЃА рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃЏрЃЌрЃџрЃўрЃљрЃюрЃЮрЃЉрЃљ.

### Core Components
* **Generator:** рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃћрЃџрЃўрЃљ рЃАрЃбрЃўрЃџрЃўрЃА рЃбрЃарЃљрЃюрЃАрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃќрЃћ (Photo $\to$ Monet).
* **Discriminator:** **PatchGAN** рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃљрЃцрЃљрЃАрЃћрЃЉрЃА рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃ«рЃљрЃарЃўрЃАрЃ«рЃА $70 \times 70$ рЃърЃљрЃбрЃЕрЃћрЃЉрЃќрЃћ (local patches) рЃЊрЃљ рЃљрЃарЃљ рЃЏрЃЌрЃџрЃўрЃљрЃю рЃАрЃБрЃарЃљрЃЌрЃќрЃћ, рЃарЃљрЃф рЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃА рЃбрЃћрЃЦрЃАрЃбрЃБрЃарЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљрЃА.
* **Loss Functions:**
    * **Adversarial Loss (LSGAN):** рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃњрЃћрЃюрЃћрЃарЃўрЃарЃћрЃЉрЃБрЃџрЃў рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃарЃћрЃљрЃџрЃўрЃАрЃбрЃБрЃарЃЮрЃЉрЃљрЃА.
    * **Cycle Consistency Loss:** рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА, рЃарЃЮрЃЏ $F(G(x)) \approx x$. рЃћрЃА рЃљрЃарЃўрЃА рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃА рЃерЃћрЃАрЃљрЃюрЃљрЃарЃЕрЃБрЃюрЃћрЃЉрЃџрЃљрЃЊ.
    * **Identity Loss:** рЃБрЃќрЃарЃБрЃюрЃЋрЃћрЃџрЃДрЃЮрЃцрЃА рЃцрЃћрЃарЃЌрЃљ рЃърЃљрЃџрЃўрЃбрЃарЃўрЃА рЃерЃћрЃюрЃљрЃарЃЕрЃБрЃюрЃћрЃЉрЃљрЃА (Monet $\to$ Monet рЃБрЃюрЃЊрЃљ рЃЊрЃљрЃарЃЕрЃћрЃА рЃБрЃфрЃЋрЃџрЃћрЃџрЃў).

## Experiments

рЃЕрЃљрЃбрЃљрЃарЃЊрЃљ рЃЮрЃарЃў рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃў, рЃАрЃљрЃЊрЃљрЃф рЃерЃћрЃўрЃфрЃЋрЃљрЃџрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ **Generator Architecture**, рЃарЃљрЃЌрЃљ рЃерЃћрЃњрЃЋрЃћрЃцрЃљрЃАрЃћрЃЉрЃўрЃюрЃљ рЃЏрЃўрЃАрЃў рЃњрЃљрЃЋрЃџрЃћрЃюрЃљ рЃАрЃбрЃўрЃџрЃўрЃА рЃбрЃарЃљрЃюрЃАрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃќрЃћ. рЃДрЃЋрЃћрЃџрЃљ рЃАрЃ«рЃЋрЃљ рЃ░рЃўрЃърЃћрЃарЃърЃљрЃарЃљрЃЏрЃћрЃбрЃарЃў (Learning Rate, Loss Weights, Normalization) рЃЊрЃљрЃарЃЕрЃљ рЃБрЃфрЃЋрЃџрЃћрЃџрЃў.

### Experiment 1: ResNet-9 Generator (Baseline)
рЃърЃўрЃарЃЋрЃћрЃџ рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃерЃў рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћрЃЌ **ResNet Based Generator** (Johnson et al.), рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃўрЃДрЃћрЃюрЃћрЃЉрЃА **Residual Blocks**-рЃА рЃЊрЃљ **Bottleneck** рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљрЃА.

* **Technical Specs:** 9 Residual Blocks, Instance Normalization, Reflection Padding.
* **Results:** рЃЏрЃЮрЃЊрЃћрЃџрЃЏрЃљ рЃљрЃЕрЃЋрЃћрЃюрЃљ рЃЎрЃљрЃарЃњрЃў рЃерЃћрЃЊрЃћрЃњрЃў "рЃЏрЃ«рЃљрЃбрЃЋрЃарЃБрЃџрЃў" рЃАрЃбрЃўрЃџрЃўрЃА рЃњрЃљрЃЊрЃЏрЃЮрЃфрЃћрЃЏрЃљрЃерЃў. **Residual Connections** рЃЉрЃЮрЃЌрЃџрЃюрЃћрЃЎрЃерЃў (Bottleneck) рЃљрЃўрЃФрЃБрЃџрЃћрЃЉрЃА рЃЦрЃАрЃћрЃџрЃА рЃўрЃАрЃгрЃљрЃЋрЃџрЃЮрЃА рЃАрЃћрЃЏрЃљрЃюрЃбрЃўрЃЎрЃБрЃарЃў рЃЏрЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃџрЃћрЃЉрЃў (semantic features), рЃарЃљрЃф рЃАрЃбрЃўрЃџрЃўрЃА рЃдрЃарЃЏрЃљ рЃбрЃарЃљрЃюрЃАрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃўрЃФрЃџрЃћрЃЋрЃљ.
* **Visual Analysis:** рЃЏрЃўрЃдрЃћрЃЉрЃБрЃџрЃў рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў рЃ«рЃљрЃАрЃўрЃљрЃЌрЃЊрЃћрЃЉрЃљ рЃЏрЃЎрЃЋрЃћрЃЌрЃарЃў рЃЏрЃЮрЃюрЃћрЃА brushstrokes-рЃўрЃЌ, рЃЌрЃБрЃЏрЃфрЃљ рЃќрЃЮрЃњрЃ»рЃћрЃа рЃерЃћрЃўрЃюрЃўрЃерЃюрЃћрЃЉрЃљ рЃЏрЃфрЃўрЃарЃћ рЃњрЃћрЃЮрЃЏрЃћрЃбрЃарЃўрЃБрЃџрЃў рЃЊрЃћрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃћрЃЉрЃў.

### Experiment 2: U-Net Generator
рЃЏрЃћрЃЮрЃарЃћ рЃћрЃЦрЃАрЃърЃћрЃарЃўрЃЏрЃћрЃюрЃбрЃерЃў рЃњрЃљрЃЏрЃЮрЃЋрЃўрЃДрЃћрЃюрЃћрЃЌ **U-Net** рЃљрЃарЃЦрЃўрЃбрЃћрЃЦрЃбрЃБрЃарЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃ«рЃљрЃАрЃўрЃљрЃЌрЃЊрЃћрЃЉрЃљ **Skip Connections**-рЃўрЃЌ рЃћрЃюрЃЎрЃЮрЃЊрЃћрЃарЃАрЃљ рЃЊрЃљ рЃЊрЃћрЃЎрЃЮрЃЊрЃћрЃарЃА рЃерЃЮрЃарЃўрЃА.

* **Technical Specs:** U-Net with Skip Connections, Instance Normalization.
* **Hypothesis:** рЃЋрЃћрЃџрЃЮрЃЊрЃўрЃЌ, рЃарЃЮрЃЏ U-Net рЃБрЃЎрЃћрЃЌ рЃерЃћрЃўрЃюрЃљрЃарЃЕрЃБрЃюрЃћрЃЉрЃЊрЃљ рЃцрЃЮрЃбрЃЮрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃА (рЃЎрЃЮрЃюрЃбрЃБрЃарЃћрЃЉрЃА), рЃЏрЃљрЃњрЃарЃљрЃЏ рЃерЃћрЃАрЃљрЃФрЃџрЃЮрЃљ рЃерЃћрЃАрЃБрЃАрЃбрЃћрЃЉрЃБрЃџрЃўрЃДрЃЮ рЃЏрЃ«рЃљрЃбрЃЋрЃарЃБрЃџрЃў рЃћрЃцрЃћрЃЦрЃбрЃў.
* **Results:** рЃ░рЃўрЃърЃЮрЃЌрЃћрЃќрЃљ рЃњрЃљрЃЏрЃљрЃарЃЌрЃџрЃЊрЃљ. **Skip Connections**-рЃЏрЃљ рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃњрЃљрЃЊрЃљрЃўрЃбрЃљрЃюрЃљ input рЃАрЃБрЃарЃљрЃЌрЃўрЃА рЃЏрЃљрЃдрЃљрЃџрЃў рЃАрЃўрЃ«рЃерЃўрЃарЃўрЃА рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў (high-frequency details) output-рЃерЃў.
* **Visual Analysis:** рЃАрЃБрЃарЃљрЃЌрЃћрЃЉрЃў рЃФрЃљрЃџрЃўрЃљрЃю рЃ░рЃњрЃљрЃЋрЃА рЃЮрЃарЃўрЃњрЃўрЃюрЃљрЃџ рЃцрЃЮрЃбрЃЮрЃћрЃЉрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃљрЃЎрЃџрЃўрЃљрЃЌ рЃўрЃА "рЃљрЃЉрЃАрЃбрЃарЃљрЃЦрЃбрЃБрЃџрЃў" рЃћрЃцрЃћрЃЦрЃбрЃў, рЃарЃљрЃф рЃЏрЃЮрЃюрЃћрЃА рЃюрЃљрЃ«рЃљрЃбрЃћрЃЉрЃА рЃљрЃ«рЃљрЃАрЃўрЃљрЃЌрЃћрЃЉрЃЌ. рЃЏрЃЮрЃЊрЃћрЃџрЃЏрЃљ рЃърЃарЃўрЃЮрЃарЃўрЃбрЃћрЃбрЃў рЃЏрЃўрЃљрЃюрЃўрЃГрЃљ **Pixel-level accuracy**-рЃА рЃЊрЃљ рЃљрЃарЃљ **Stylistic abstraction**-рЃА.

## Conclusion & Analysis

рЃерЃћрЃЊрЃљрЃарЃћрЃЉрЃўрЃЌрЃЏрЃљ рЃљрЃюрЃљрЃџрЃўрЃќрЃЏрЃљ рЃљрЃЕрЃЋрЃћрЃюрЃљ, рЃарЃЮрЃЏ **Style Transfer** рЃљрЃЏрЃЮрЃфрЃљрЃюрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА **ResNet Generator** рЃБрЃцрЃарЃЮ рЃћрЃцрЃћрЃЦрЃбрЃБрЃарЃўрЃљ, рЃЋрЃўрЃЊрЃарЃћ U-Net.

рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ Submission-рЃўрЃАрЃЌрЃЋрЃўрЃА рЃЊрЃљ **MiFID** рЃЏрЃћрЃбрЃарЃўрЃЎрЃўрЃА рЃњрЃљрЃАрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃћрЃЉрЃџрЃљрЃЊ рЃљрЃарЃЕрЃћрЃБрЃџ рЃўрЃЦрЃюрЃљ **Experiment 1 (ResNet)** рЃЏрЃЮрЃЊрЃћрЃџрЃў.
